# Eventos Chu - Plataforma de Gestión Premium

Plataforma de alto rendimiento diseñada para la gestión integral de eventos y giras de **Hyenuk Chu**. Este ecosistema combina una experiencia de usuario fluida con un panel administrativo de nivel empresarial para el control total de inscripciones en tiempo real.

## 🚀 Stack Tecnológico de Vanguardia

- **Next.js 14 (App Router)**: Arquitectura moderna para un rendimiento óptimo.
- **GSAP (GreenSock)**: Motor de animaciones de alto nivel para transiciones premium y micro-interacciones.
- **Supabase & Realtime**: Base de datos relacional con sincronización instantánea vía WebSockets.
- **Clerk Auth**: Gestión de identidad y seguridad de usuarios de última generación.
- **Upstash Redis**: Capa de caché ultra-rápida para la entrega inmediata de eventos.
- **Keap (Infusionsoft)**: Integración profunda con CRM para automatización de marketing.
- **Tailwind CSS & Shadcn/UI**: Sistema de diseño basado en tokens para una consistencia visual absoluta.

## ✨ Características Principales

### 🖥️ Admin Dashboard (Command Center)
- **Métricas Live**: Visualización en tiempo real de Usuarios Registrados, Inscripciones Totales, Solicitudes Pendientes y Aprobadas.
- **Buscadores Inteligentes**: Sistema unificado de búsqueda `Popover + Command` para filtrar eventos, categorías y estados con precisión quirúrgica.
- **Gestión Granular**: Control total sobre cada inscripción, permitiendo editar datos del usuario y estados de asistencia por evento.
- **Diseño Globalizado**: Interfaz profesional basada en tokens de diseño neutros para una experiencia libre de distracciones.

### 🛡️ Seguridad y Robustez
- **Sincronización de Identidad**: Detección automática de cambios de perfil en Clerk para mantener la integridad de los datos en Supabase.
- **Protección Anti-Bot**: Integración nativa con Cloudflare Turnstile en todos los formularios de registro.
- **Sistema de Caché Inteligente**: Los eventos se sirven desde Redis para eliminar la latencia de la base de datos principal.

### 🔔 Notificaciones Inteligentes
- **Alertas en Tiempo Real**: Notificaciones visuales inmediatas cuando un nuevo usuario se registra o completa su formulario de perfil.
- **Feedback Operativo**: Confirmaciones visuales para cada acción administrativa (duplicado de eventos, cambios de estado, etc.).
